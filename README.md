# Trading Signal Telegram Bot v0.2

Все настройки, кроме токена и админа, вынесены в Telegram-бота.

## Railway Variables

```env
BOT_TOKEN=токен_бота_из_BotFather
```

Опционально, чтобы бот отвечал только тебе:

```env
ALLOWED_USER_IDS=твой_telegram_id
```

## Кнопки

- `/signal` — общий скан всех монет
- `new` — подсказка для добавления монеты
- `exit` — подсказка для удаления монеты
- `⚙️ Настройки` — все настройки бота
- `🖼 Визуализация` — режим отправки графика и текста
- `Азия / Америка` — сессионный фильтр
- `Ping`
- `Сброс`

## Команды

```text
/signal
/signal BTC
/viz
/settings
/myid
new pol
exit sol
```

## Что делает /signal

Без аргумента `/signal` сканирует все монеты из списка и показывает лучшие сигналы.
Для полного сигнала с графиком используй `/signal BTC` или просто напиши `BTC`.

## Визуализация

Добавлена кнопка `🖼 Визуализация` и команда `/viz`.

Режимы:

- **График сверху, текст снизу** — сначала отдельная картинка графика, потом полный текст сигнала.
- **Текст сверху, график снизу** — сначала полный текст сигнала, потом отдельная картинка графика.
- **Всё в одной картинке** — бот пытается собрать график и текст в одну картинку.
- **Авто** — режим по умолчанию: бот пробует отправить всё одной картинкой; если текст слишком длинный или картинка получается неудобной, отправляет график отдельной картинкой, а ниже текст с сигналом. Если график не построился, отправляет только текст.

## Важно

Это алгоритмический анализ по свечам MEXC. Это не финансовая рекомендация и не гарантия прибыли.

## v0.2 update

Добавлена визуализация сигнала:

- новая кнопка `🖼 Визуализация` в клавиатуре бота;
- новое меню выбора режима;
- режим `Авто` по умолчанию;
- fallback: если картинка не собирается, бот всё равно отправляет сигнал текстом.

## v0.1 update

В полном сигнале добавлены:
- **Main TF success** — успешность выбранного основного таймфрейма.
- **Total success** — общий процент по всем таймфреймам.
- Основной таймфрейм в блоке успешности выделяется ⭐ и жирным шрифтом.

## v0.1 update: 1M timeframe

Месячный таймфрейм оставлен как старший тренд-фильтр.
Для `1M` бот теперь требует не 80 свечей, а минимум 24.
Если MEXC не отдаёт нативные месячные свечи, бот строит синтетические 1M свечи из недельных свечей.
